Question 1

What is the aspect ratio of this geographic region?

BBOX width = (maxx) - (minx) = (-106.5)- (-109.5) = 3 degrees
BBOX height = (maxy) - (miny) = (36.5) - (34.5) = 2 degrees
BBOX aspect ratio = (BBOX width) / (BBOX height) = (3) / (2) = 1.5

Question 2

What would be the height (in whole pixels) for a map image for thisregion that is 600 pixels wide?

aspect ratio = width / height = 200 / height = 1.5
height = (width) / (aspect ratio) = 600 / 1.5 = 400 pixels
