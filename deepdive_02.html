Deep Dive 2

Question 1

What request type is common across all three (WMS, WFS, WCS) OGC web services that we have learned about?

"GetCapabilities" is common across all three (WMS,WFS,WCS) OGC web services that we have studied.

Question 2

2.1 What are the required parameters, and what do they represent? 2.2 What is returned in response to a WMS GetCapabilities request?

2.1: The required parameters are "SERVICE=WMS" and "REQUEST=GetCapabilities" and they represent "service type" and "request name".

2.2: The GetCapabilities request returns an XML file that provides the capabilities of the service.

Question 3

What are the required parameters, and what do they represent? 3.2 What is returned in response to a WMS GetMap request? 3.3 What is the significance of transparency in WMS requests?

  • VERSION=version
  • REQUEST=GetMap
  • LAYERS=layer_list
  • STYLES=style_list
  • SRS=namespace:identifier
  • BBOX=minx,miny,maxx,maxy
  • WIDTH=output_width
  • HEIGHT=output_height
  • FORMAT=output_format

3.2: It will return "Map Images", which are graphic files representing layers from a WMS Service. If the bounding box is incorrect all you will recieve is a blank page.

3.3: Transparency allows for the viewing of multiple layers at the same time. If this is not allowed then only the top layer will be present and it will block the layers lower and inside.

Question 4

What OGC request would you use to inform the configuration of a client application (like ArcGIS) about an OGC service that you want to add layers from?

The "GetCapabilites" request is used to inform the configuration of a client application (like ArcGIS) about an OGC service that you could add layers from.

Question 5- Which OGC request would you submit under the following circumstances (include both the service type [e.g. WMS, WFS, WCS], and the request [e.g. GetMap, GetCapabilities, GetCoverage, etc.] in your answer)

5.1 You want a map image representing three layers of data in a single JPEG for a specified area of interest. 5.2 You want to retrieve data representing geometries and associated attributes for a road network, with the returned data in GML. 5.3 You want to retrieve data representing a digital elevation model (a raster dataset) in the form of a GeoTIFF.

5.1: WMS GetMap request.

5.2: WFS GetFeature request.

5.3: WCS GetCoverage request.
